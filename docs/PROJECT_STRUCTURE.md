# Sanabel OS Project Structure

Sanabel OS هي منصة تشغيل فقط وليست نظامًا ماليًا موحدًا.

كل مؤسسة أو برنامج داخل مجلد apps مستقل تمامًا:
- بيانات مستقلة
- مستخدمون مستقلون
- صلاحيات مستقلة
- سجل تجاري مستقل
- رقم ضريبي مستقل
- حسابات مالية مستقلة

المنصة الرئيسية index.html وظيفتها فتح البرامج فقط.

المسارات الحالية:
- منصة Sanabel OS: /index.html
- مؤسسة حسن الشريمي: /apps/alshuraimi/index.html
- برنامج الذمم: /apps/receivables/index.html
- برنامج الكاشير: /apps/cashier/index.html

مجلد shared يحتوي أدوات واجهة مشتركة فقط، ولا يحتوي بيانات لأي مؤسسة.
مجلد config يحتوي إعدادات عامة للمنصة فقط، وليس إعدادات المؤسسات.
