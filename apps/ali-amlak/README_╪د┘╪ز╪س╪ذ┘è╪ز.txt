أملاك علي يحيى الشريمي وأولاده V1.4

الجديد:
1) شاشة إدارة النظام.
2) تنزيل واستعادة نسخة احتياطية.
3) مسح بيانات العمل مع إبقاء المستخدمين وإعدادات المؤسسة.
4) حماية المسح بكلمة مرور المدير وكتابة كلمة "حذف".
5) تجهيز الربط السحابي مع Supabase.
6) النسخة الجديدة تبدأ دون محلات أو عقود تجريبية.

خطوات الربط السحابي:
1) أنشئ مشروعًا في Supabase.
2) افتح SQL Editor والصق محتوى ملف supabase_setup.sql ثم Run.
3) من Project Settings > API انسخ Project URL و anon public key.
4) افتح البرنامج > إدارة النظام.
5) الصق الرابط والمفتاح ثم اضغط حفظ واختبار الاتصال.
6) اضغط رفع بيانات هذا الجهاز لأول مرة.
7) على أي جهاز آخر افتح نفس النسخة، أدخل نفس الرابط والمفتاح، ثم اضغط تنزيل البيانات من السحابة.

مهم:
- خذ نسخة احتياطية قبل المسح أو قبل استبدال البيانات السحابية.
- النشر على GitHub Pages أو Netlify يجعل البرنامج متاحًا من رابط ويتيح تثبيته على الجوال.
